Instalando no MacOS:

{% highlight bash %}
$ brew tap homebrew/science
$ brew install r
{% endhighlight %}


Legal vamos fazer uns testes, e para isso iniciaremos um terminal R:

{% highlight bash %}
$ R
{% endhighlight %}


Sintax e comandos básicos do ``**R**``

{% highlight bash %}

//Atribuir valor a uma variavel

x <- 5
x <- 3+2
x <- (3+2)*(21)
x //Saída 105

x <- TRUE
x //Saída TRUE

{% endhighlight %}
